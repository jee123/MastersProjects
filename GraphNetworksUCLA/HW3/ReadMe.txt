Requirements

--------------------------------------------------------------------

Following are the system requirements for the execution of this script :



1) Windows 7/8, Mac OSX 10.6+
2) R
3) RStudio
3) Intel Processor
4) igraph,netrw, Matrixstats library


-----------------------------Configuration

--------------------------------------------------------------------

R version 2.15.3 is preferred
------------------------------Installation

--------------------------------------------------------------------

To run the script, please follow the below steps:



Mac OSX :


1) Please make sure R is installed on the machine.

2) Open terminal.

3) Go in the directory where the script has been saved.

4) Run the below command in order.

	

chmod 777 HW3_104588717_904588105_004589213.R
	./HW3_104588717_904588105_004589213.R




Windows :


1) Please make sure R and igraph are installed on the machine.

2) Open RStudio.

3) Open the script (HW3_104588717_904588105_004589213.R) in RStudio.

4) Run the code.
